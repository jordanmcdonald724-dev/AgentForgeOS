# AgentForgeOS Studio (Phase 6 Scaffold)

This folder contains the minimal Studio interface scaffold defined in the documentation. It follows the five-region layout from `docs/UI_STUDIO_LAYOUT.md`:

1. Top Navigation Bar
2. Left Sidebar
3. Main Workspace
4. Agent Console
5. Pipeline / System Monitor

The included `index.html` and `style.css` render a static mock of the required layout to keep phase progression unblocked. Hook this scaffold to a build pipeline (e.g., Vite/React) when implementing interactive functionality.
