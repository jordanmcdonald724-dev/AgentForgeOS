Welcome to AgentForgeOS

This wizard will guide you through configuring your AI operating system. You'll set up your LLM, image, and voice providers along with storage and filesystem access.

All values are saved to config/.env on the host. You can re-run this wizard at any time from Settings.

LLM Provider

Choose the large language model provider that agents will use. Ollama runs fully locally. OpenAI requires an API key.

Your key is stored locally in config/.env and is never sent anywhere else.

Image & Voice Providers

Select providers for AI image generation and text-to-speech output. These can be changed later.

Image generation
Text-to-speech

Storage & Bridge

The Bridge is the sandboxed filesystem layer that agents use to read and write project files. Set the root directory it's allowed to access. MongoDB is optional — leave blank to run in-memory.

Agents cannot read or write outside this directory.

Review & Save

Double-check your settings below, then click Save & Launch to write them to config/.env and open the Studio.

Bootstrap dependencies

Run backend dependency installation directly from the wizard. You can optionally install the desktop shell (npm) as well.
